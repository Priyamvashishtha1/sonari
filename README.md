# SonariCalc Mobile

Cross-platform mobile version of the gold calculator, rebuilt in React Native with Expo for Android and iOS.

## Included features

- Live gold price calculation
- Karat switching
- Making charges, discount, and GST sliders
- Owner and customer views
- Dark and light themes
- Saveable local session history
- Native mobile share sheet
- Sticky final price footer

## Run locally

```bash
npm install
npm start
```

Then launch directly on an emulator/device with:

```bash
npm run android
npm run ios
```

If you want a full native build instead of the standard Expo flow:

```bash
npm run android:native
npm run ios:native
```
